# Информационная система ветеринарной клиники — Frontend

Одностраничное приложение на базе SvelteKit для информационной системы ветеринарной клиники.

## Технологический стек

- **Фреймворк:** [SvelteKit](https://kit.svelte.dev/) с TypeScript
- **Стилизация:** Vanilla CSS + utility-классы
- **Иконки:** [Lucide Svelte](https://lucide.dev/)
- **Экспорт в Excel:** [SheetJS (xlsx)](https://sheetjs.com/)

## Разработка

Установите зависимости и запустите dev-сервер:

```bash
npm install
npm run dev
```

Приложение ожидает, что бэкенд API запущен по адресу `http://localhost:8080`.  
При необходимости настройте базовый URL в файле `src/lib/api/http.ts`.

## Сборка

```bash
npm run build
npm run preview   # предпросмотр продакшен сборки локально
```

## Структура проекта

```text
src/
├── lib/
│   ├── api/          # HTTP-клиент и обертки для API
│   ├── components/   # Переиспользуемые Svelte-компоненты
│   ├── stores/       # Состояние на базе Svelte 5 runes
│   └── types.ts      # Общие интерфейсы TypeScript
└── routes/           # Файловая маршрутизация SvelteKit
    ├── app/          # Защищенные маршруты приложения
    └── login/        # Страница авторизации
```
