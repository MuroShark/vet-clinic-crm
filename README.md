# Информационная система ветеринарной клиники

Полностековая система управления ветеринарной клиникой, созданная на базе ASP.NET Core и SvelteKit.

## Возможности

- **Регистратура** — запись на прием, реестр клиентов и пациентов, управление ежедневной очередью
- **Ветеринар** — личное расписание, медицинские карты пациентов, управление диагнозами и лечением
- **Главный врач** — расписание всей клиники, управление услугами и ценами, справочник диагнозов
- **Директор** — отчеты по зарплате на основе KPI, финансовая аналитика, управление запасами
- **Администрирование** — учетные записи сотрудников, роли пользователей (RBAC), управление филиалами, журнал аудита
- **Склад** — партии товаров, отслеживание сроков годности, списание при закрытии приема

## Технологический стек

| Уровень     | Технология                          |
|-------------|-------------------------------------|
| Backend     | ASP.NET Core 8, Entity Framework Core, MS SQL Server |
| Frontend    | SvelteKit (Svelte 5), TypeScript, Vanilla CSS |
| Авторизация | JWT (HMAC-SHA256), хеширование паролей PBKDF2 |
| Деплой      | Docker Compose, nginx (TLS), Let's Encrypt |

## Структура проекта

```text
.
├── AhillBackend/        # ASP.NET Core Web API
│   ├── Auth/            # Сервис JWT токенов и хеширование паролей
│   ├── Controllers/     # API контроллеры и DTO мапперы
│   ├── Data/            # EF Core DbContext и сидер базы данных
│   ├── Dtos/            # Request / response DTOs
│   └── Models/          # Доменные модели
├── ahill-frontend/      # SvelteKit SPA
│   └── src/
│       ├── lib/
│       │   ├── api/         # Клиент API бэкенда
│       │   ├── components/  # Svelte компоненты (представления по ролям)
│       │   └── stores/      # Реактивное состояние приложения (Svelte 5 runes)
│       └── routes/          # Файловая маршрутизация SvelteKit
├── docker-compose.yml           # Стек для локальной разработки
├── docker-compose.prod.yml      # Продакшен стек с nginx + certbot
└── DEPLOY.md                    # Руководство по развертыванию
```

## Быстрый старт (Локально)

### Требования

- .NET 8 SDK
- Node.js 20+
- Docker (для базы данных) или локальный экземпляр MS SQL Server

### Backend

```bash
cp .env.example .env      # настройте строку подключения и настройки JWT
cd AhillBackend
dotnet run
```

При первом запуске база данных создается автоматически и заполняется демо-данными.

### Frontend

```bash
cd ahill-frontend
npm install
npm run dev
```

Приложение доступно по адресу `http://localhost:5173`.
